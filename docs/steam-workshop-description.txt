Let Claude or ChatGPT run your beaver colony. Timberbot API gives AI full read/write access to every game system over HTTP -- it reads your beavers, places buildings, manages crops, and keeps everyone alive while you watch.

[code]
/loop 1m /timberbot keep my beavers alive and get to 77 wellbeing
[/code]

That's it. One line in Claude Code and the AI plays Timberborn for you -- checking water before drought, planting crops, building housing, unlocking science, managing power grids, and assembling bots.

Or use it as a scripting API for your own tools, dashboards, and automation.

[h1]What can the AI see and do?[/h1]

[b]Every beaver and bot:[/b] position on the map, which district they belong to, all 38 wellbeing needs by category, what they're carrying, workplace, contamination status. Bot energy, grease, and deterioration.

[b]Every building:[/b] workers, power state, production progress, inventory, recipes, construction status, effect radius. Place and demolish buildings with full game-native validation.

[b]The whole map:[/b] terrain height, water depth, badwater contamination, soil moisture, every occupant on every tile (including vertical stacking). ASCII visual with height shading.

[b]Colony management:[/b] worker assignment, priorities, crops, tree marking, science unlocks, floodgates, stockpiles, distribution, breeding, clutch control, work schedule, game speed.

[b]Power networks:[/b] which buildings share a power grid, supply vs demand per network, generator output.

[b]One-call snapshot:[/b] entire colony in a single request -- resources, weather, drought countdown, alerts, food/water/log/plank/gear day projections.

[h1]How it works[/h1]

Timberbot API runs an HTTP server (port 8085) inside the game. Python client included:

[code]
timberbot.py summary                              # full colony snapshot
timberbot.py beavers detail:full                  # every need on every beaver
timberbot.py power                                # power networks with supply vs demand
timberbot.py visual x:120 y:140 radius:10         # ASCII map
timberbot.py place_building prefab:FarmHouse.IronTeeth x:120 y:130 z:2
[/code]

Works with any language that speaks HTTP. Python client is optional -- raw curl works too.

[h1]AI setup (Claude Code)[/h1]

[code]
cp docs/timberbot.md ~/.claude/skills/timberbot/SKILL.md
/loop 1m /timberbot
[/code]

The mod ships with an AI prompt that teaches Claude every game mechanic, API command, and building recipe. The AI decides its own strategy -- you just tell it what you want.

Works with ChatGPT too -- paste docs/timberbot.md as a system prompt.

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game -- API starts automatically on port 8085
3. pip install requests toons
4. Run: timberbot.py ping
